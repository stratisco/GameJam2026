## Rooms

### Weird Puzzle Room
This room is based off the XOR function which toggles between 1 and 0.

This room contains a puzzle with a 5x7 grid of bulbs. Each bulb can either be ON or OFF and the goal is to turn every bulb ON

There is a button for each row and for each collumn. Pressing a button flips the state of specific lights in its row/collumn. 

The buttons however do not affect every bulb in their row/collumn and instead will only flip specific bulbs. Players must find the correct combination of buttons to turn each light on.

Example: If the button for collumn 1 is pressed the 2nd, 3rd and 5th bulb in collumn 1 will flip their state.

*This puzzle UI can be ran by 'instantiating' the `res://weirdPuzzle/weirdBoard.tscn` to another scene (atm its in the testing scene)*
